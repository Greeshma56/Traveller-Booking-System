# ✈️ Traveller Booking System

## 📌 Description
This is a responsive traveller booking web application developed using HTML, CSS, and JavaScript. It allows users to explore destinations, view images, and book trips.

## 🚀 Features
- View travel destinations (Goa, Kerala, Manali)
- Image gallery for each location
- Book trips with date selection
- View booking history
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html
- booking.html
- booking-history.html
- confirm.html
- trips.html
- gallery pages
- script.js
- style.css
- images
